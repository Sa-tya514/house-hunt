# house-hunt
- HouseHunt is a web and mobile app built with the MERN stack (MongoDB, Express.js, React.js, Node.js).
- It helps users search, rent, buy, or list properties easily.
- Users can filter homes by location, price, size, and amenities.
- Listings include photos, videos, floor plans, and 360° virtual tours.
- The app provides market insights like price trends and neighborhood details.
- Renters get alerts for new listings or price drops.
- Owners can add, edit, and manage properties after admin approval.
- Admins oversee the platform, approve owners, and enforce rules.
- The backend uses Express.js + MongoDB for data storage and APIs.
